# despachante
> Traduce las politicas de usuarios en comandos que el sistema operativo puede reconocer

## Instalación

```sh
python setup.py install
```

## Configuracion

Editar archivo `/etc/netcop/netcop.config`

## Ejecución

```sh
$ despachar
```

## Ejecutar pruebas

```sh
python setup.py tests
```
